# myproject

Reusable Python project template with:

- CLI support
- FastAPI API
- Webhook endpoint
- Tests with pytest
- Linting with ruff
- Type checking with mypy
- pre-commit hooks
- GitHub Actions CI

## Requirements

- Docker Desktop
- WSL2
- VS Code
- Dev Containers

## Local development

Open the project in the Dev Container, then run:

```bash
pip install -e .
pip install -r requirements.txt
